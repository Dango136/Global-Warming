# Global-Warming
This TG Bot will explain what is Global Warming and what we can do to slow it down or even prevent it!
## Technologies:
+ Python
+ TeleBot library
## Functionality:
1. The bot must tell the user about Global Warning, what is causing it and how we can slow it.
2. The user can choose what exactly they want to know about Global Warming.
3. The user can use buttons to choose the theme.
## Commands:
+ ```/start``` - The bot will show user the greeting message.
+ ```/help``` - Shows the full list of commands.
+ ```/gw``` - The bot will show a set of few buttons, each one offering the user to get the answer to the specific question.
